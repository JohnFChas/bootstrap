Skapa en ny hemsida med AngularJS och Bootstrap
Hemsidan ska ha:
- En bootstrap Navbar, gärna som en partial
- En main controller i index
- En "Home" view med en bootstrap Carousel
- En "About" view med en bootstrap Jumbotron
- En valfri view som listar något sorts objekt
	- Varje objekt i listan ska använda bootstrap columner så att du får rader med objekt i listan

Tänk på att alla vyer bör använda bootstrap columner (till exempel col-md-12), även de som använder hela bredden.
Man vill inte ha en massa element som ligger ensamma utanför bootstraps grid-struktur.

Användbara länkar:
Bootstrap components: http://getbootstrap.com/components/
Bootstrap på w3schools: http://www.w3schools.com/bootstrap/default.asp
